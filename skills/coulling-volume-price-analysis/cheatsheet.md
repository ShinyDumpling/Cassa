# 交易决策速查表

## 量价关系判断

| 价格变动 | 成交量 | 判断 | 行动 |
|---------|--------|------|------|
| 大涨 | 高 | 确认，真实上涨 | 持有多头/考虑加仓 |
| 大涨 | 低 | 异常，潜在陷阱 | 警惕，等待后续确认 |
| 小涨 | 高 | 异常，市场弱势 | 考虑减仓/准备反转 |
| 小涨 | 低 | 确认，正常波动 | 维持现有头寸 |
| 大跌 | 高 | 确认，真实下跌 | 持有空头/考虑加仓 |
| 大跌 | 低 | 异常，潜在陷阱 | 警惕，等待后续确认 |
| 小跌 | 高 | 异常，市场强势 | 考虑减仓/准备反转 |
| 小跌 | 低 | 确认，正常波动 | 维持现有头寸 |

## 阶段识别

| 阶段 | 价格行为 | 成交量特征 | 局内人行为 |
|------|---------|-----------|-----------|
| 吸筹 | 反复下跌反弹，低位震荡 | 下跌放量，反弹缩量 | 利用利空低价买入 |
| 拉升 | 平稳上涨，小幅回落 | 成交量平稳放大 | 持有并推高价格 |
| 派筹 | 反复上涨回落，高位震荡 | 上涨放量，回落缩量 | 利用利好高价卖出 |
| 下跌 | 平稳下跌，小幅反弹 | 成交量平稳放大 | 持有空头/等待吸筹 |

## 测试信号

| 测试类型 | 条件 | 成功信号 | 失败信号 |
|---------|------|---------|---------|
| 供给测试 | 吸筹后，价格回落至前期卖压区 | 低成交量 → 可做多 | 高成交量 → 继续震仓 |
| 需求测试 | 派筹后，价格回升至前期买压区 | 低成交量 → 可做空 | 高成交量 → 继续派筹 |

## 趋势反转预警信号

**顶部反转信号（按严重程度递增）：**
1. 射击十字星 + 低成交量 → 短期停滞
2. 射击十字星 + 高成交量 → 弱势初现
3. 连续射击十字星 + 成交量递增 → 抛售高峰
4. 短阳线 + 高成交量 → 出货确认
5. 放量止涨 → 顶部确认

**底部反转信号（按严重程度递增）：**
1. 锤头线 + 低成交量 → 短期止跌
2. 锤头线 + 高成交量 → 买盘初现
3. 连续锤头线 + 成交量递增 → 买入高峰
4. 短阴线 + 高成交量 → 吸筹确认
5. 放量止跌 → 底部确认

## 局内人行为判断

| 如果你看到 | 局内人可能在 |
|-----------|------------|
| 利空消息 + 价格暴跌 + 高成交量后止跌 | 吸筹（利用恐慌低价买入） |
| 利好消息 + 价格持续上涨 + 顶部窄幅震荡 | 派筹（利用贪婪高价卖出） |
| 价格回到前期密集区 + 低成交量 | 测试（确认市场余量） |
| 价格回到前期密集区 + 高成交量 | 测试失败（需要更多准备） |

## 持仓管理规则

- **当确认持续时**：持有头寸，不要因小幅波动离场
- **当出现第一次异常时**：提高警惕，不要立即行动。等待后续K线确认
- **当出现第二次异常（同方向）时**：减仓50%
- **当出现第三次异常或放量止涨/止跌时**：全部了结头寸
- **当测试成功 + 突破 + 放量时**：建立新头寸的最佳时机
