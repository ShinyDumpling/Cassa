# 第八章：动态趋势及趋势线

## 核心观点
忘掉传统的趋势线。当你按照传统方式画出趋势线想要进场时，聪明的资本恰恰在清仓离场。动态趋势线让你在趋势形成初期就进场，而非在趋势末尾。

## 关键概念

- **传统趋势线的弊端**：基于历史高/低点连线，滞后于市场，往往在趋势接近尾声时才给出信号
- **动态趋势线**：结合量价分析，基于价格行为动态调整，而非机械连接高/低点
- **趋势早期进场**：通过量价分析识别吸筹结束和测试成功，在趋势启动前建立头寸

## 关键收获

1. 传统趋势线会让你在趋势末期进场，成为局内人的接盘侠
2. 动态趋势线 + 量价分析 = 在趋势形成初期进场
3. 趋势的确认来自成交量，而非线段的连接
