# 第五章：量价分析的全局视角

## 核心观点
吸筹和派筹是量价分析的宏观框架，出现在任何时间框架和任何市场。局内人通过两种情绪（恐惧和贪婪）操纵市场，完成低价吸筹→推高价格→高价派筹的循环。测试是判断行动是否完成的关键信号。

## 框架介绍

### 五大核心概念

- **吸筹 (Accumulation)**
  - 何时用：市场经历长期下跌后，出现反复震荡的价格区间
  - 如何用：识别价格反复下跌反弹、每次下跌都在"震仓"的形态。局内人通过利空消息制造恐慌，迫使散户卖出，自己低价买入
  - 特征：多次下跌→反弹的循环，成交量在下跌时放大，每次反弹都较弱。最终以抛售高峰结束

- **派筹 (Distribution)**
  - 何时用：市场经历长期上涨后，出现顶部震荡区间
  - 如何用：识别价格在顶部反复震荡、无法继续上涨的形态。局内人通过利好消息制造贪婪，吸引散户追高，自己高价卖出
  - 特征：多次上涨→回落的循环，成交量在上涨时放大，每次回落都有更多买家涌入。最终以买入高峰结束

- **供给测试 (Supply Test)**
  - 何时用：吸筹阶段结束后，价格回到之前卖压区时
  - 如何用：局内人将价格压低至先前有大量卖盘的价格区间，观察成交量
  - **成功**（低成交量）：卖盘已被全部吸收，可以推高市场
  - **失败**（高成交量）：仍有卖盘残留，需要继续震仓

- **需求测试 (Demand Test)**
  - 何时用：派筹阶段结束后，价格回到之前买压区时
  - 如何用：局内人将价格推高至先前有大量买盘的价格区间，观察成交量
  - **成功**（低成交量）：买盘已被全部满足，可以推低市场
  - **失败**（高成交量）：仍有买盘，需要继续派筹

- **抛售高峰 (Selling Climax) / 买入高峰 (Buying Climax)**
  - 何时用：吸筹/派筹阶段的终点
  - 如何用：抛售高峰是恐慌性卖出的顶点，成交量急剧放大后市场止跌。买入高峰是贪婪性买入的顶点，成交量急剧放大后市场见顶。两者都是"烟火秀"——旧趋势的终点和新趋势的起点

### 局内人操纵的两个情绪杠杆

- **恐惧** → 制造利空消息 → 市场暴跌 → 散户恐慌卖出 → 局内人低价吸筹
- **贪婪** → 制造利好消息 → 市场持续上涨 → 散户追高买入 → 局内人高价派筹

## 关键概念

- **局内人 (Insiders)**：做市商、大型操盘手、专业投资者——拥有信息优势和资金优势的市场参与者
- **震仓 (Shakeout)**：局内人通过制造价格波动迫使散户卖出的手段，如同摇晃树干收获果实
- **批发价 vs 零售价**：局内人以批发价买入（吸筹），以零售价卖出（派筹），赚取差价
- **媒体是终极武器**：任何消息——自然灾害、政治声明、经济数据、财务报表——都为操纵市场提供了借口

## 心理模型

- 把市场想象成批发商的仓库运作：填满仓库（吸筹）→ 发起营销活动推高需求（拉升）→ 清空仓库（派筹）→ 重复
- 局内人像是在摇晃树干收获果实：反复震荡直到所有"顽固的果实"都掉落
- 测试就像市场调研：在全面推广前，先小规模测试市场反应

## 反模式

- **在吸筹阶段追高买入**：你买的是局内人震仓后的小反弹，他们在收集筹码，你却在帮他们抬价
- **在派筹阶段追涨**：局内人正在清仓，你是最后的接盘侠
- **忽视测试信号**：成功的低成交量测试是最强烈的趋势即将启动的信号之一

## 实例演练

**场景**：某股票在经历数月下跌后，在某个价格区间反复震荡，出现了三次"下跌→反弹"的循环。第三次下跌时成交量明显放大，但价格没有再创新低。

**分析**：这是典型的吸筹阶段。局内人通过三次震仓，已经将大部分"顽固"的卖方洗出。第三次下跌的高成交量 + 未创新低 = 抛售高峰信号。随后若出现一次低成交量的下跌测试（供给测试），确认卖盘已枯竭，则市场即将上涨。

**行动**：等待供给测试成功后，在价格突破盘整区间时建立多头头寸。

## 关键收获

1. 吸筹和派筹是所有市场的永恒循环，存在于任何时间框架
2. 测试是局内人发出的最清晰信号——低成交量测试成功 = 趋势即将启动
3. 局内人操纵的不是市场，而是交易者的情绪（恐惧和贪婪）
4. 成交量无法隐藏——局内人的每一步行动都会在成交量上留下痕迹
5. 抛售高峰和买入高峰是"烟火秀"——旧趋势的终点，新趋势的起点
