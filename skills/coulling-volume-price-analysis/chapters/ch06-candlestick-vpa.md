# 第六章：结合K线图的量价分析

## 核心观点
K线图比柱状图承载更多信息。结合K线图的量价分析是作者的个人方法，通过K线形态和成交量的双重验证，可以更深入地了解市场行为。射击十字星和锤头线是最重要的两种K线形态。

## 框架介绍

### K线图分析五原则

1. **影线优先**：上影线或下影线的长度是首要因素，它们立刻显示市场的强势、弱势或迟疑，更重要的是显示市场情绪
2. **无影线 = 强势**：没有影线表明朝着收盘价方向的市场情绪强势
3. **实体 = 情绪强度**：低实体 = 市场情绪弱，高实体 = 市场情绪强
4. **位置决定意义**：同一类型的K线图在趋势中不同位置有完全不同的含义。必须结合更长趋势和盘整阶段考虑
5. **成交量验证价格**：先看K线图，再看成交量柱状图——确认还是异常

### 三种最重要的K线形态

#### 射击十字星 (Shooting Star) — 弱势信号
- **形态**：上影线长，实体小，收于开盘价附近。买方推动价格走高后被卖方压制回落
- **出现位置**：上涨趋势中 → 潜在顶部反转；下跌趋势中 → 弱势确认
- **量价结合**：
  - 低成交量 + 射击十字星 → 短期停滞/回调
  - 高成交量 + 射击十字星 → 强势弱信号，可能反转
  - 连续2-3个射击十字星 + 成交量递增 → 看跌情绪指数级增强，可能构成抛售高峰
- **在下跌趋势中**：可能是局内人在反弹中卖出存货（需求测试），确认市场仍处弱势

#### 锤头线 (Hammer) — 强势信号
- **形态**：下影线长，实体小，收于开盘价附近。卖方推动价格走低后被买方吸收反弹
- **出现位置**：下跌趋势中 → 潜在底部反转
- **量价结合**：
  - 单根锤头线 + 高成交量 → 止跌信号，但不足以确认反转
  - 连续2-3根锤头线 + 高成交量 → 买入高峰信号，趋势反转确认
- **测试中的锤头线**：吸筹完成后，价格回测时出现低成交量锤头线 = 供给测试成功

#### 放量止跌 / 放量止涨 (Stopping Volume)
- **放量止跌**：下跌趋势中，某根K线伴随极高成交量，但价格未再创新低。抛售压力被买方大量吸收，是底部信号
- **放量止涨**：上涨趋势中，某根K线伴随极高成交量，但价格未再创新高。买入压力被卖方大量吸收，是顶部信号

## 关键概念

- **多重时间跨度确认**：小时图上的射击十字星，结合15分钟图上的双射击十字星 + 高成交量，确认反转可能性
- **K线图位置的重要性**：刚从吸筹阶段脱离的市场不太可能出现反转；刚完成测试的突破更可信
- **价格盘整的深度与长度**：盘整越久，突破或反转越可信

## 心理模型

- 射击十字星 = 买方"冲锋"失败，被卖方击退——市场疲软
- 锤头线 = 卖方"砸盘"失败，被买方吸收——市场强势
- 连续出现的同一形态 = 情绪的指数级累积——不是加法，是乘法

## 反模式

- **看到锤头线就立即买入**：市场在暴跌后不会立即反转，锤头线可能只是局内人"强行买入"制造的暂时止跌，后续还有下跌
- **仅靠价格行为（PAT）交易**：没有成交量，你不知道射击十字星是真实的弱势还是做市商的陷阱
- **忽视K线在趋势中的位置**：同一形态在趋势顶部、中部、底部的含义完全不同

## 实例演练

**场景**：日线图上，某股票经历两周上涨后，出现一个射击十字星，伴随高于平均水平的成交量。随后第二天又出现一个射击十字星，成交量比第一天更高。

**分析**：第一根射击十字星——弱势初始信号，提醒提高警惕。第二根射击十字星 + 更高成交量——弱势确认，卖盘力量在增强。两个射击十字星在同一价格区域出现，成交量递增 → 抛售高峰的可能性增加。

**行动**：如果持有多头，逐步减仓。不要立即做空——等待第三个确认信号（如低实体阴线 + 高成交量，或跌破支撑位）后再考虑做空。

## 关键收获

1. 影线是第一优先指标，它直接反映市场情绪
2. 射击十字星和锤头线是最重要的K线形态，但必须结合成交量
3. 连续出现的同一形态，成交量递增 → 信号强度指数级增强
4. K线图在趋势中的位置不同，含义完全不同
5. 多重时间跨度分析是确认信号的有效工具
