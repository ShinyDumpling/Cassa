# 第十一章：综合运用

## 核心观点
将所有量价分析工具整合为一个完整的交易体系。横盘整理的价格形态在结合成交量后变得极其强大，提供简单明了的交易机会——前提是你有足够的耐心。

## 框架介绍

### 完整交易流程

1. **确定大趋势**：使用多重时间跨度判断主要趋势方向（日线/周线）
2. **识别吸筹/派筹阶段**：寻找价格盘整区间，观察成交量模式
3. **等待测试信号**：这是最关键的一步——低成交量测试成功 = 行动计划即将启动
4. **确认突破**：价格突破盘整区间 + 成交量放大 = 进场信号
5. **持仓管理**：持续监控量价关系，确认 = 持有，异常 = 减仓
6. **离场信号**：反向信号出现 + 量价确认 → 了结头寸

### 横盘整理形态的威力

- **低位横盘**：吸筹阶段，多次下跌测试后成交量递减 → 买入信号
- **高位横盘**：派筹阶段，多次上涨测试后成交量递减 → 卖出信号
- **横盘长度与后续趋势**：盘整越久（因），突破后趋势越大（果）——威科夫因果定律

### 多重时间跨度策略

- **长周期定方向**：周线/日线判断主要趋势
- **中周期找机会**：小时图识别吸筹/派筹和测试
- **短周期精进场**：15分钟/5分钟图精确入场时机

## 心理模型

- 交易如猎人狩猎：大部分时间在等待，只在最佳时机出手
- 多重时间跨度 = 望远镜 + 放大镜：长周期看方向，短周期看细节

## 关键收获

1. 完整交易流程：趋势→盘整→测试→突破→持仓→离场
2. 耐心是量价分析交易者最重要的品质
3. 横盘整理 + 成交量 = 局内人的行动地图
4. 多重时间跨度提供三维视角
