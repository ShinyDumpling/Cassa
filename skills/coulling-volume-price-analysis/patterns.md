# 量价分析模式汇总

## 威科夫量价分析框架
**何时用**：每次分析图表时
**如何使用**：供求定律判断方向 → 因果定律判断规模 → 投入产出定律判断真实性
**权衡**：需要主观判断，无法自动化；需要持续练习

## 吸筹-派筹循环
**何时用**：识别市场的主要阶段转换
**如何使用**：吸筹（低位震荡，下跌放量）→ 拉升（成交量平稳放大）→ 派筹（高位震荡，上涨放量）→ 下跌（成交量平稳放大）→ 循环
**权衡**：不同市场吸筹/派筹的时间跨度差异巨大（数小时到数月）

## 确认 vs 异常模式
**何时用**：分析每一根K线图
**如何使用**：
- 长阳线 + 高成交量 → 确认，上涨真实
- 长阳线 + 低成交量 → 异常，可能多头陷阱
- 短阳线 + 低成交量 → 确认，小波动正常
- 短阳线 + 高成交量 → 异常，市场弱势，出货信号
**权衡**：单一异常信号不足以确认反转，需要后续K线验证

## 供给/需求测试模式
**何时用**：价格回到前期重要价格区间时
**如何使用**：
- 供给测试：价格下跌至前期卖压区，低成交量 → 成功，可做多
- 需求测试：价格上涨至前期买压区，低成交量 → 成功，可做空
- 任何测试失败（高成交量）→ 局内人需要更多准备时间
**权衡**：测试是最强的信号之一，但需要耐心等待

## 射击十字星连续模式
**何时用**：上涨趋势中连续出现射击十字星时
**如何使用**：第一根 → 提高警惕；第二根 + 成交量更高 → 弱势确认；第三根 → 抛售高峰可能，准备反转
**权衡**：必须结合趋势位置和支撑/阻力位判断

## 锤头线连续模式
**何时用**：下跌趋势中连续出现锤头线时
**如何使用**：第一根 → 止跌信号；第二根 + 成交量更高 → 买盘增强；第三根 → 买入高峰可能，准备反转
**权衡**：单根锤头线不足以确认反转，暴跌后通常还有下跌

## 多重时间跨度确认模式
**何时用**：单一时间框架信号不明确时
**如何使用**：长周期定方向 → 中周期找机会 → 短周期精进场。三个时间跨度的信号一致时，胜率最高
**权衡**：需要同时监控多个图表，增加分析复杂度

## 横盘整理突破模式
**何时用**：价格在区间内盘整后
**如何使用**：横盘（吸筹/派筹）+ 低成交量测试成功 + 突破 + 成交量放大 = 进场
**权衡**：盘整越久，突破越可靠；但假突破仍可能发生，需要成交量确认

## 动态趋势线模式
**何时用**：替代传统趋势线
**如何使用**：基于量价关系动态调整趋势线，在趋势形成初期而非末期进场
**权衡**：需要更多主观判断，不如传统趋势线直观
